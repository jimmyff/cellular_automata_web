Cellular Automata web demos

cellular_automata is an open source dart library by jimmyff for exploring & playing with cellular automata.

  • Github repositories:
    • https://github.com/jimmyff/cellular_automata
      • https://github.com/jimmyff/cellular_automata_web
  • Pub packages:
    • https://pub.dartlang.org/packages/cellular_automata
    • https://pub.dartlang.org/packages/cellular_automata_web

Basic demo

This is a very basic playing Conway's game of life. It's configured with button controls to play/pause/step and it resets when the scene is becomes stable.
